# 二次元小说网站 - 产品需求文档

## 1. 产品概述
基于 Cloudflare Workers 构建的二次元风格小说阅读网站，依托 R2 对象存储和 KV 存储提供高性能的小说存储、在线阅读和多格式下载服务。
- 面向二次元爱好者和轻小说读者的轻量化阅读平台
- 利用 Cloudflare 边缘计算能力提供全球低延迟访问体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 访问方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览小说列表、在线阅读、下载小说、搜索小说 |

### 2.2 功能模块
1. **首页**：导航栏、轮播推荐、分类筛选、热门小说列表、最新更新
2. **小说详情页**：小说信息、封面展示、章节列表、在线阅读入口、下载按钮
3. **阅读页**：章节内容展示、阅读设置（字体、主题、字号）、章节导航、进度保存
4. **搜索页**：关键词搜索、标签筛选、分类浏览

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 顶部导航栏 | Logo展示、搜索框、导航菜单、深色/浅色主题切换 |
| 首页 | Hero轮播区 | 展示推荐小说、自动轮播、点击跳转详情 |
| 首页 | 分类筛选栏 | 按分类（玄幻、恋爱、校园、奇幻、科幻等）筛选小说 |
| 首页 | 小说列表 | 卡片式展示小说封面、标题、作者、简介、最新章节 |
| 小说详情页 | 信息展示区 | 小说封面、标题、作者、分类、标签、简介、字数统计 |
| 小说详情页 | 章节列表 | 卷章分组、章节标题、更新时间、阅读进度标记 |
| 小说详情页 | 下载区域 | 多格式下载按钮（TXT、EPUB、PDF）、格式说明 |
| 阅读页 | 阅读区域 | 章节标题、正文内容、段落优化排版 |
| 阅读页 | 阅读控制栏 | 字号调整、行间距、主题切换（日间/夜间/护眼/粉色） |
| 阅读页 | 章节导航 | 上一章/下一章、目录侧边栏、阅读进度条 |
| 阅读页 | 进度记忆 | 使用 KV 存储用户阅读进度、自动跳转上次位置 |
| 搜索页 | 搜索框 | 实时搜索建议、搜索历史 |
| 搜索页 | 搜索结果 | 匹配小说列表、结果高亮展示 |

## 3. 核心流程

### 3.1 阅读流程
用户访问首页 → 浏览/搜索小说 → 点击进入详情页 → 选择章节 → 进入阅读页 → 阅读并调整设置 → 进度自动保存 → 切换下一章或返回

### 3.2 下载流程
用户进入小说详情页 → 选择需要的格式（TXT/EPUB/PDF）→ 点击下载 → Workers 从 R2 获取文件 → 提供下载响应

## 4. 用户界面设计

### 4.1 设计风格
- **二次元风格**：柔和的粉色/紫色渐变作为主色调，配合可爱的装饰元素
- **主色调**：樱花粉 (#FFB7C5)、薰衣草紫 (#E6E6FA)、天蓝 (#87CEEB)
- **辅助色**：奶白 (#FFFAF0)、深灰 (#2D2D2D)、樱红 (#FF6B8A)
- **按钮风格**：圆角胶囊型按钮，带有微动效和渐变，hover 时有缩放和发光效果
- **字体选择**：
  - 标题：ZCOOL KuaiLe（可爱风格中文字体）
  - 正文：Noto Serif SC（优雅的阅读字体）
  - UI文字：Nunito（圆润的无衬线字体）
- **布局风格**：卡片式布局，大量留白，柔和阴影，毛玻璃效果
- **图标风格**：Lucide React 图标，配合可爱的 emoji 点缀
- **装饰元素**：樱花飘落动画、星星闪烁、圆角设计、柔和渐变背景

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero轮播区 | 大尺寸轮播图，小说封面+标题叠加，渐变遮罩，自动切换动效 |
| 首页 | 小说卡片 | 圆角卡片，封面图悬停放大，阴影加深，标签彩色徽章 |
| 首页 | 导航栏 | 毛玻璃效果，固定顶部，滚动时背景变化 |
| 小说详情页 | 封面区域 | 大封面带阴影，信息区渐变背景，下载按钮醒目 |
| 小说详情页 | 章节列表 | 折叠面板，已读章节标记，hover高亮 |
| 阅读页 | 阅读区域 | 模拟书本排版，舒适行间距，纸张质感背景 |
| 阅读页 | 设置面板 | 侧边滑出，选项卡式设计，滑块控件 |
| 全站 | 背景 | 渐变背景 + 樱花飘落粒子动画 |

### 4.3 响应式设计
- 桌面端优先设计，最大宽度 1400px 居中布局
- 平板端自适应：两列变单列，导航收起
- 移动端优化：汉堡菜单，触摸友好按钮，阅读区域全屏优化
- 触摸手势支持：左右滑动切换章节

### 4.4 支持的小说格式
- **在线阅读**：解析 TXT/EPUB 内容为 HTML 展示
- **下载格式**：TXT、EPUB、PDF 三种格式
- R2 存储原始文件，Workers 按需处理格式转换（如需要）
